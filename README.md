# Fusion_

Fusion_, aynı klasörde bulunan birden fazla XML dosyasını tek bir çıktı dosyasında birleştirmek için hazırlanmış basit bir konsol uygulamasıdır.

## Ne Yapar

- kullanıcıdan XML dosyalarının bulunduğu klasörü alır
- seçilen sayı kadar XML dosyasını okur
- verileri birleştirip `Merge.xml` olarak yazar

## Teknolojiler

- C#
- .NET
- `DataSet`
- XML

## Notlar

- Uygulama işlem öncesinde klasördeki XML dosyalarını yeniden adlandırır
